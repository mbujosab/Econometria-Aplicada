Este repositorio (rama [[https://github.com/mbujosab/Econometria-Aplicada/tree/main][=main=]]) contiene el código fuente para la generación del material de clase de un curso sobre series temporales.

El material está escrito en formato [[https://orgmode.org/][Org Mode]] (utilizando la configuración [[https://github.com/jkitchin/scimax][Scimax]] de [[https://www.gnu.org/software/emacs/][GNU Emacs]], desarrollada por el [[https://engineering.cmu.edu/directory/bios/kitchin-john.html][Profesor John Kitchin]]).

Las lecciones están escritas en formato de [[https://github.com/jkitchin/scimax-eln][cuadernos electrónicos]] (véase también [[https://github.com/jkitchin/scimax/blob/master/scimax-notebook.org][scimax-notebook.org]]) y se exportan a =pdf= y =html=, también como presentaciones para el aula y como Notebooks de [[https://jupyter.org/try][Jupyter]] (usando [[https://github.com/jkitchin/ox-ipynb][ox-ipynb]]).

Los documentos en formato =pdf= son los más completos. El lenguaje de programación utilizado en los Notebooks de Jupyter es [[https://www.python.org/][Python]]. Las prácticas están diseñadas para ser realizadas con [[https://gretl.sourceforge.net/es.html][Gretl]].

El código de este repositorio se compila mediante GitHub Actions, y el material generado se publica en la rama [[https://github.com/mbujosab/Econometria-Aplicada/tree/gh-pages][=gh-pages=]] de este repositorio. Los detalles sobre cómo se compila el material y se publica pueden consultarse en el documento [[https://github.com/mbujosab/Econometria-Aplicada/blob/main/org-publish.org][=org-publish.org=]]. 

Los Notebooks de Jupyter generados durante la compilación se pueden ejecutar en línea en MyBinder accediendo al siguiente enlace: [[https://mybinder.org/v2/gh/mbujosab/Econometria-Aplicada/https://mybinder.org/v2/gh/mbujosab/Econometria-Aplicada/gh-pages?labpath=CuadernosElectronicos/gh-pages?labpath=CuadernosElectronicos/]].

Todo el material compilado es accesible desde la url [[https://mbujosab.github.io/Econometria-Aplicada/]].
