La rama [[https://github.com/mbujosab/Econometria-Aplicada/tree/main][main]] de este contiene el código fuente para la generación del material de clase de un curso sobre series temporales.

El material, ya compilado, es accesible desde la url [[https://mbujosab.github.io/Econometria-Aplicada/]].

Este material está escrito en ficheros con formato [[https://orgmode.org/][Org Mode]] (utilizando como editor [[https://www.gnu.org/software/emacs/][GNU Emacs]] con la configuración [[https://github.com/jkitchin/scimax][Scimax]] desarrollada por el [[https://engineering.cmu.edu/directory/bios/kitchin-john.html][Profesor John Kitchin]]); 
y constituye un curso de series temporales con modelización ARIMA.

Las lecciones están escritas en formato de [[https://github.com/jkitchin/scimax-eln][cuadernos electrónicos]] (véase [[https://github.com/jkitchin/scimax/blob/master/scimax-notebook.org][scimax-notebook.org]]). 
Son exportadas a =pdf= y =html=, y también como presentaciones en formato [[https://revealjs.com/][reveal.js]] para ser proyectadas el aula durante la clase. 
También son exportadas como Notebooks interactivos de [[https://jupyter.org/try][Jupyter]] (usando [[https://github.com/jkitchin/ox-ipynb][ox-ipynb]]).

- Los documentos en formato =pdf= son los más completos.
- El lenguaje de programación utilizado en los Notebooks de Jupyter es [[https://www.python.org/][Python]].
- Pero las prácticas del curso están diseñadas para ser ejecutadas con [[https://gretl.sourceforge.net/es.html][Gretl]].

El código de este repositorio se compila mediante GitHub Actions; y el material generado se publica en la rama [[https://github.com/mbujosab/Econometria-Aplicada/tree/gh-pages][gh-pages]] de este repositorio. 
Los detalles sobre cómo se compila el material y se publica pueden consultarse en el documento [[https://github.com/mbujosab/Econometria-Aplicada/blob/main/org-publish.org][org-publish.org]]. 

Los Notebooks de Jupyter generados durante la compilación se pueden ejecutar en línea en MyBinder accediendo al siguiente enlace: 
[[https://mybinder.org/v2/gh/mbujosab/Econometria-Aplicada/gh-pages?urlpath=%2Fdoc%2Ftree%2FCuadernosElectronicos]]
