Este repositorio contiene el código fuente para la generación del material de clase de un curso de series temporales.

El material está escrito en formato [[https://orgmode.org/][Org Mode]], utilizando la configuración [[https://github.com/jkitchin/scimax][Scimax]] de [[https://www.gnu.org/software/emacs/][GNU Emacs]], desarrollada por el [[https://engineering.cmu.edu/directory/bios/kitchin-john.html][Profesor John Kitchin]].

El código de este repositorio se compila mediante GitHub Actions, y el material generado se publica en la rama =gh-pages= de este repositorio. Los detalles sobre cómo se compila el material y se publica en la rama =gh-pages= se pueden consultar en el documento =org-publish.org=. Los Notebooks de Jupyter generados durante la compilación se pueden ejecutar en línea en MyBinder accediendo al siguiente enlace: [[https://mybinder.org/v2/gh/mbujosab/Econometria-Aplicada/gh-pages?labpath=CuadernosElectronicos/]].

Todo el material publicado es accesible en [[https://mbujosab.github.io/Econometria-Aplicada/]].
