Curso de series temporales para la asignatura Econometría Aplicada

Grado en Economía de la UCM

La rama main de este contiene el código fuente para la generación del material de clase de un curso sobre series temporales.

El material, ya compilado, es accesible desde la url https://mbujosab.github.io/Econometria-Aplicada/.

Este material está escrito en ficheros con formato Org Mode (utilizando como editor GNU Emacs con la configuración Scimax desarrollada por el Profesor John Kitchin); y constituye un curso de series temporales con modelización ARIMA.

Las lecciones están escritas en formato de cuadernos electrónicos (véase scimax-notebook.org). Son exportadas a pdf y html, y también como presentaciones en formato reveal.js para ser proyectadas el aula durante la clase. También son exportadas como Notebooks interactivos de Jupyter (usando ox-ipynb).

  • Los documentos en formato pdf son los más completos.
  • El lenguaje de programación utilizado en los Notebooks de Jupyter es Python.
  • Pero las prácticas del curso están diseñadas para ser ejecutadas con Gretl.

El código de este repositorio se compila mediante GitHub Actions; y el material generado se publica en la rama gh-pages de este repositorio. Los detalles sobre cómo se compila el material y se publica pueden consultarse en el documento org-publish.org.

Los Notebooks de Jupyter generados durante la compilación se pueden ejecutar en línea en MyBinder accediendo al siguiente enlace: https://mybinder.org/v2/gh/mbujosab/Econometria-Aplicada/gh-pages?urlpath=%2Fdoc%2Ftree%2FCuadernosElectronicos
