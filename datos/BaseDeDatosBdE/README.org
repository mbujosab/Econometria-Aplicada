
Base de datos del Banco de España para instalación en Gretl.

- Fecha de descarga: 30 de agosto de 2025.

/Puede instalar una versión actualizada directamente desde la aplicación de Gretl; por lo tanto, estos archivos pueden no ser útiles./

Coloco estos archivos aquí porque Gretl solo permite la instalación de bases de datos a través de su interfaz gráfica, y no se puede acceder a dicha interfaz mediante las Actions de GitHub. Sin embargo, las Actions de GitHub me permiten acceder a este repositorio para copiar estos archivos en la instalación de Gretl dentro de la carpeta ~/.gretl/db/.
